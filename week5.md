# 빅데이터

## 빅데이터의 이해

### 빅데이터의 의미
  - 대량의 정형 반정형 비정형 데이터로부터 의미 있는 가치를 추출하고 유용한 결과를 도출하는 기술
  - 통상 적으로 사용하는 데이터 소프트 웨어의 수용 한계를 넘어서는 엄청난 용량의 데이터를 의미

### 빅데이터의 속성(V3)
  - 규모(Volume)
  - 다양성(Variety)
  - 속도(Velocity)
  - 외 신뢰성, 가치정확성, 휘발성 등이 추가

## 빅데이터 처리 과정
  - 수집
  - 저장
  - 처리
  - 분석
  - 표현

## 빅데이터 기술
  - 데이터 수집 기술
    - 데이터 수집 기술은 다양한 유형의 정형 반정형 비정형 데이터를 추찰하고 모으는 기술
    - 빅 데이터 서비스의 품질을 좌우하는 주요한 기술
    - 로그수집기, 크롤링, 센싱, RSS리더/오픈API,ETL 등
  - 데이터 저장 기술
    - 분산 파일 시스템, NoSQL, 병렬DBMS, 네트워크 구성 저장 시스템
  - 빅데이터 처리 기술
    - 전처리 방식
    - 실시간 처리 방식
    - 대표적인 빅데이터 처리 기술
      - 맵리듀스: 구글의 소프트웨어 프레임 워크
      - 하둡: 오픈소스로 구현한 프레임워크
      - 빅데이터 처리에 활용: R, NoSQL, 카산드라등
      - 빅데이터 처리 기술은 사용 솔루션보다 하둡을 포함한 오픈소스의 비중이 높음
    - 빅데이터 분석기술
      - 통계 분석
        - 기술 통계, 다변량 분석, 회귀분석, 시계열 분석
      - 데이터 마이닝
        - 목표지향
          - 분류, 추정, 예측
        - 목표불명
          - 군집화, 연관성, 표현
      - 연관성 분석 기술
      - 협업 필터링
      - 텍스트 마이닝
        - 문서요약, 문서분류, 문서군집, 특성추출
